About the Data

Our dataset contains all federally declared disasters in the United States from 1953 to 2021. The original dataset is from fema.gov but we used a refined version from user "Heads or Tails" on Kaggle.com.

Modifications to Original Dataset

We removed the following columns from the dataset: disaster_closeout_date, place_code, declaration_request_number, last_refresh, and hash.

Licensing

The data was retrieved on 4/16/21. This product uses the Federal Emergency Managemen Agency's API, but is not endorsed by FEMA.
